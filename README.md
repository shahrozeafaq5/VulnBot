## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/VulnBot.git
cd VulnBot
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure environment

(Create your `.env` file...)

### 4. Run

```bash
python main.py
```
